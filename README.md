# BloodLink - Blood Management System

## Project Description

BloodLink is a web-based Blood Management System developed using Laravel and MySQL.

The system helps manage:

- Blood requests
- Donor registration
- Blood inventory
- Serology testing
- Blood issuing
- Patient tracking
- Admin dashboard

This project is being developed as part of CS 476 - Software Development.
